Modern, app-like lesson interface for TutorLMS with responsive design and clean navigation.

== Description ==
Transform your TutorLMS lessons into a modern, engaging learning environment...

== Installation ==
1. Download the theme ZIP file from GitHub releases
2. Go to Appearance > Themes > Add New > Upload Theme
3. Activate the theme

== Requirements ==
* TutorLMS 3.0+
* Hello Elementor parent theme
